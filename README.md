# Banking Simulator Frontend

This repository contains the **frontend** for the Banking Simulator project.  
It is built using **React** and communicates with the Spring Boot backend via REST APIs.

## Features
- User login and authentication
- View account details
- Perform transactions (deposit, withdraw, transfer)
- View alerts and notifications

## Technologies Used
- React
- HTML, CSS, JavaScript
- Axios (for API calls)

## Prerequisites
- Node.js (v14+ recommended)
- npm or yarn
- Backend running at `http://localhost:8080` (Spring Boot backend)

## How to Run

   ```bash
   git clone https://github.com/your-username/BankingSimulator1-frontend.git

cd BankingSimulator1-frontend
